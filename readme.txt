Allows commenters on an entry to subscribe to e-mail notifications for subsequent comments. This is a fork of the popular Subscribe to Comments plugin.


== Description ==

Subscribe to Comments enables commenters to sign up for e-mail notification of subsequent entries. The plugin includes a full-featured subscription manager that your commenters can use to unsubscribe to certain posts, block all notifications, or even change their notification e-mail address!
